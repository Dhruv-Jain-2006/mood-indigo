# mood-indigo
An emotion-driven travel planner website which helps you travel according to your mood.
Mood Indigo is a cinematic, minimal web experience that helps users explore travel destinations based on how they feel, not just where they want to go. It blends storytelling, motion, and calm interaction design into a smooth single-page journey.
